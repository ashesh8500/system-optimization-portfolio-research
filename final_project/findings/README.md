# Findings

Each subfolder should describe one coherent experiment or study line.

- **Template:** [`_template/FINDINGS.md`](_template/FINDINGS.md)
- **Master list:** [`INDEX.md`](INDEX.md)

Create a new folder:

```bash
mkdir -p findings/<run_id>
cp findings/_template/FINDINGS.md findings/<run_id>/FINDINGS.md
```

Then edit `FINDINGS.md`, add artifacts, and register the run in `INDEX.md`.
